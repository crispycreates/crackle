Step 1 · Pick your engine
Choose how The Crackle finds and scores leads: pure reasoning, browser, hybrid, or raw search.
Mode aware inputs
Agent
Pure reasoning
Pros
  • Fast and simple.
  • Great for lead criteria, scoring logic, and cold email drafts.
Cons
  • No live verification.
  • Can hallucinate if you ask it to invent leads.
Use your strongest reasoning model, like GPT‑5.4.
Agent + Computer Use
Browser native
Pros
  • Inspects real websites for Meta Pixel and Google tags.
  • Can follow multi‑step navigation and funnels.
Cons
  • Slower and more brittle when sites block automation.
  • Needs careful prompting and more compute.
Use a tool‑capable agent (GPT‑5.4 + browser tools).
Perplexity + OpenAI
Hybrid research
Pros
  • Fresh, citation‑backed live web data.
  • OpenAI handles synthesis and scoring.
Cons
  • More moving parts and integration work.
  • Overkill for tiny lists.
Perfect when you care about freshness and citations.
OpenAI Only
Pure generation
Pros
  • Fastest to wire up.
  • Great for copy, subject lines, and reasoning.
Cons
  • No live verification.
  • No direct pixel or tag checks.
Run GPT‑5.4 or similar for best quality.
Two‑Stage
Plan → execute
Pros
  • Separates plan from execution for complex searches.
  • Easier to debug and refine.
Cons
  • Slower than one‑shot.
  • More state and code to manage.
Use best reasoning model for planning, faster for execution.
Web Search Only
Raw discovery
Pros
  • Fast, predictable keyword prospecting.
  • Great for quick scans and manual review.
Cons
  • Weakest synthesis.
  • Needs extra step for pixel checks.
Anchor to search, then synthesize in a second pass.
